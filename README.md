Cílem této práce je vytvořit aplikaci pro analýzu a validaci HTML dokumentů. Program dokáže načíst HTML kód od uživatele nebo ze souboru, rozložit ho na jednotlivé syntaktické jednotky, zkontrolovat formální správnost tagů, detekovat nepovolené tagy, ověřit správné párování a vnoření tagů a vyhodnotit četnost použití jednotlivých elementů. Výsledky jsou zobrazovány v přehledném formátu s informací o řádku, na kterém se případná chyba nachází. Projekt je implementován v jazyce C# s využitím WPF pro grafické uživatelské rozhraní a GitHub pro verzování a sdílení kódu. Tento nástroj demonstruje praktické využití syntaktické analýzy, regulárních výrazů a datových struktur při validaci webových dokumentů.
